Model Building

After comparing eight machine learning algorithms, the site decided to use this algorithm to predict.

Random Forest

Based on machine learning algorithms, the system can efficiently diagnose the presence of heart problems and allow users to get results in real-time. For Ramdom Forest, its accuracy is 80.328%. 

Background

Random Forests is a supervised learning model that are primarily used for classification and regression. It is randomly built and consists of many decision trees and is integrated to get accurate results. Unlike decision tree, random forest segmentation features and the process of finding root nodes are random. In a random forest, there is no connection between each decision tree. Because each decision tree exists alone, the predicted results are often inaccurate, so after they are brought together, the overall prediction function will strengthen.

Reality Difficulty

Real-world applications are indeed more complex, and the differences between different classification errors can be quite large. For example, in medical diagnosis, cancer is regarded as a positive category, and health is regarded as a negative category. Then predicting a sick patient healthy as a false negtive is more serious than predicting a patient who is not ill as a false positive.

Confusion Matrix 

It is a table used to describe the performance of the classifier, with four variables. TP(True Positive) is predicted yes and actual yes. FN(False Negative) is predicted no and actual yes, and also called ”Type II error”. 

Among them, yes is a disease, no is no disease. As shown in the Figure, it needs to make FN as small as possible; that is, the Type II error is small. Then, KNN and SVM should not be the most appropriate classifier because the possibility of missed diagnosis is big. Combined with the accuracy, Random Forest is a suitable classifier.


ROC Curve

The ROC curve is a plot of the TPR(True Positive Rate) for the y-axis, and the FPR(False Positive Rate) for the x-axis according to different thresholds.

Generally, the default threshold is 0.5. If the probability is greater than 0.5, then the sample is divided into positive sample by the model, and vice versa.

AUC is the area under curve, the larger the value of AUC, the better. As shown in the Figure, AUC is 0.896. When its value is greater than 0.80, it is a excellent discrimination.
