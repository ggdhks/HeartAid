Data Analysis

Data analysis aims to manipulate, clean, and process data in Python.

Dataset Source

The dataset source is from the UCI Machine Learning Repository. It has 303 records, of which 139 has heart disease, and the rate of illness is 45.87%.

Read More

Missing Value

The original data set has missing values and redundant attributes.  The data source provided on Kaggle has solved this problem.

Among the 13 attributes, 6 attributes are numeric values and there are 7 categorical values. The main point is to encode the categorical value, so the method of one-hot coding is adopted.
